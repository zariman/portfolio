Who Will Win March Madness? HTML/CSS, JavaScript, Python, Django

This project helps users visualize which stat categories contribute the most in winning a NCAA tournament game. Users can use this information to predict who will have the upperhand between any two NCAA teams based on their regular season statistics (work in progress).

Github Run Space Pong HTML/CSS, JavaScript, Java, Spring Boot

This project utilizes sockets to enable an online multiplayer experience between two players in a game of classic pong. Graphics have been upgraded to 3-D with the help of Three.js. Try your luck in becoming the best pong player in the universe! (work in progress)

Github Run 24 Seconds C#, Unity3D, Blender

Developed for Android and compatible for most phones. Started out as a solo passion project. Swipe the screen to dribble and shoot a basketball. Objective is simple: put the ball in the basket. Gameplay modes include tutorial, shootaround, 3-point contest, and player vs. CPU. Available in the Google Play Market now!
